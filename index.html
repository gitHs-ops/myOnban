지금 어떤
불편함이 있나요?

카카오 채널로 매일 메뉴를 보내고 계시죠? 정말 열심히 운영하고 계신 겁니다.
하지만 이런 불편함은 없으셨나요?

📝
매일 손으로 텍스트 작성
메뉴가 많을수록 입력하는 데 시간이 걸리고, 오탈자가 생기기 쉬워요.
📞
주문 전화가 많아요
"이거 있어요?", "얼마예요?" 문의 전화에 음식 만들다 손 씻고 받기 힘드시죠.
🙈
사진이 없어서 아쉬워요
텍스트만으로는 맛있어 보이기 어렵습니다. 사진을 보면 더 사고 싶어지잖아요.
😓
품절 안내가 번거로워요
다 팔린 메뉴를 찾는 연락이 와도 일일이 답하기 쉽지 않아요.

고객 경험

고객은 어떻게
주문하나요?

스마트폰만 있으면 됩니다. 별도 앱 설치가 전혀 필요 없어요.

1
📱 카카오톡으로 메뉴 공지를 받아요
점주님이 매일 보내시는 메뉴 메시지를 고객이 확인합니다.
2
🔗 "메뉴 전체보기" 링크를 눌러요
메시지 맨 아래 링크를 누르면 주문 페이지가 열립니다.
3
🍱 사진과 가격을 보고 메뉴를 골라요
각 메뉴 사진, 가격, 품절 여부를 한눈에 확인합니다.
4
🛒 장바구니에 담고 수량을 정해요
여러 메뉴를 한 번에 담을 수 있고, 합계 금액이 자동으로 계산됩니다.
5
💰 계좌번호가 화면에 자동으로 뜨고 입금해요
국민은행 계좌번호가 자동으로 표시되고 복사 버튼 하나로 편하게 입금합니다.
6
✅ 점주님이 입금 확인 후 주문 완료!
입금 확인만 하시면 됩니다. 모든 과정이 자동으로 진행됩니다.

점주 운영 방법

점주님은 무엇을
하시면 되나요?

기존에 하시던 방식과 거의 같습니다. 딱 두 가지만 추가로 하시면 돼요.

📋
① 메뉴 관리 페이지에서 오늘 메뉴 입력
이름, 가격, 재고 수량, 사진을 등록하면 고객 페이지가 자동으로 업데이트됩니다.

기존에 쓰시던 카카오 텍스트를 붙여넣기 하면 자동으로 입력도 됩니다!
📤
② 카카오 채널에 메시지 발송
메뉴 관리 페이지에서 복사 버튼 한 번을 누르면 카카오에 붙여넣기할 텍스트가 준비됩니다.

맨 아래 링크가 자동으로 포함됩니다.

💡 기억하실 것은 딱 하나! 매일 메뉴 입력 → 복사 → 카카오에 붙여넣기.
나머지는 모두 자동입니다.

기대 효과

좋아지는 점은
무엇인가요?

점주님과 고객 모두에게 좋아집니다.

👩‍🍳 점주님 입장에서

  • 전화 주문 문의가 줄어들어 음식 만드는 데 집중할 수 있어요
  • 메뉴 작성 시간이 줄어들어요 — 카카오 텍스트 붙여넣기 한 번으로 끝
  • 사진이 있어서 더 맛있어 보이고, 구매로 이어질 가능성이 높아져요
  • 품절 메뉴 문의 전화가 줄어들어요 — 화면에 자동으로 표시되니까요
  • 지난 메뉴 기록이 자동으로 남아서 인기 메뉴를 파악할 수 있어요

👨‍👩‍👧 고객 입장에서

  • 사진을 보고 고를 수 있어서 더 믿음이 가요
  • 전화 안 해도 혼자 골라서 주문할 수 있어요
  • 어린이가 먹을 수 있는 메뉴를 한눈에 골라요
  • 장바구니에 여러 개 담고 금액을 한 번에 확인할 수 있어요
  • 계좌번호 복사 버튼 하나로 편하게 입금해요
