1

날짜 선택하기

오늘 메뉴를 입력하거나, 내일 메뉴를 미리 준비할 수 있습니다.

1
날짜 버튼 클릭
상단의 2026년 4월 19일 (오늘) 버튼을 누르면 달력이 열립니다.
2
달력에서 날짜 선택
원하는 날짜를 탭하면 그 날짜의 메뉴 화면으로 전환됩니다.
🟡 주황 점이 있는 날짜 = 이미 메뉴가 저장된 날
3
◀ ▶ 버튼으로 하루씩 이동
날짜 버튼 양쪽의 화살표로 전날·다음날로 이동할 수 있습니다.
"오늘" 버튼을 누르면 오늘 날짜로 바로 돌아옵니다.
💡 내일 메뉴를 오늘 미리 입력해 두셔도 됩니다!
고객 페이지는 오늘 날짜 메뉴를 먼저 보여주고, 없으면 가장 최근 날짜 메뉴를 자동으로 표시합니다.
2

일일 메뉴 등록하기

"일일 메뉴 등록" 카드에서 메뉴 정보를 입력하고 추가 버튼을 누릅니다.

일일 메뉴 등록
어린이도 먹을 수 있는 메뉴
추 가
1
메뉴명 입력
메뉴 이름을 입력합니다. 글자를 입력하면 과거에 쓴 메뉴 이름이 자동으로 추천됩니다. 목록에서 선택하거나 새로 입력하세요.

⚠ 이미 오늘 목록에 있는 메뉴를 입력하면 중복 경고가 표시됩니다.
2
금액 입력 (천원 단위)
숫자만 입력합니다. 예) 10,000원이면 10을 입력
3
재고 수량 입력
오늘 판매할 수량을 입력합니다. 재고가 0이면 고객 화면에 "품절"로 표시됩니다.
4
카테고리 선택
메인 반찬 김치 밀키트 기타

해당하는 종류를 고르세요.
5
어린이 가능 체크 (선택)
어린이도 먹을 수 있는 메뉴라면 체크박스에 ✓ 표시하세요.
고객 화면에 👧 표시가 됩니다.
6
추 가 버튼 클릭
버튼을 누르면 아래 메뉴 목록에 바로 추가됩니다. 입력창은 자동으로 비워집니다.
💡 메뉴명 입력 후 Enter 키를 눌러도 추가됩니다!
3

사진 등록하기

메뉴 목록에서 각 메뉴에 사진을 등록할 수 있습니다. 사진이 있으면 고객이 더 잘 구매합니다!

1
메뉴 목록에서 📷 버튼 클릭
각 메뉴 왼쪽에 있는 📷 카메라 아이콘을 누르면 사진 등록 창이 열립니다.
이미 사진이 등록된 메뉴는 아이콘 대신 작은 사진 썸네일이 표시됩니다.
2
📁 사진 선택 버튼 클릭
스마트폰 갤러리나 PC 폴더에서 사진을 고릅니다.
3
자동 업로드 완료
사진을 고르면 "업로드 중..." 메시지가 뜨고, 잠시 후 "업로드 완료!"가 표시되며 창이 닫힙니다.
이후 메뉴 목록에 사진 썸네일이 표시됩니다.
버튼기능
📁 사진 선택새 사진을 등록하거나 기존 사진을 교체합니다
삭제등록된 사진을 지웁니다
취소변경 없이 창을 닫습니다
💡 사진은 인터넷에 안전하게 저장됩니다. 다른 기기에서 열어도 사진이 그대로 보입니다.
⚠ 인터넷이 연결된 상태에서만 사진 업로드가 됩니다.
4

재고 수량 변경하기

메뉴 목록에서 수량을 바로 늘리거나 줄일 수 있습니다.

📷

💡 재고가 0이어도 고객 화면에서 메뉴가 보입니다.
품절 표시와 함께 "전화 문의 가능" 안내가 자동으로 표시됩니다.
5

금액 수정하기

메뉴 목록에서 금액을 바로 수정할 수 있습니다.

1
금액 숫자 클릭
메뉴 목록에서 10천원 을 클릭하면 입력창으로 바뀝니다.
2
새 금액 입력
숫자를 입력합니다. (천원 단위)
3
Enter 또는 다른 곳 클릭
Enter를 누르거나 다른 곳을 클릭하면 저장됩니다.
취소하려면 Esc 키를 누르세요.
6

메뉴 삭제하기

잘못 입력한 메뉴를 삭제할 수 있습니다.

1
메뉴 오른쪽 ✕ 버튼 클릭
각 메뉴 맨 오른쪽에 있는 ✕ 버튼을 누릅니다.
2
확인 창에서 "확인" 클릭
"이 메뉴를 삭제할까요?" 확인 창이 뜨면 확인을 누르면 삭제됩니다.
⚠ 삭제한 메뉴는 복구할 수 없습니다. 신중하게 눌러주세요.
7

카카오 발송 문자 복사하기

메뉴 입력이 완료되면 카카오 채널에 보낼 문자가 자동으로 만들어집니다.

1
"카카오 발송" 탭 클릭
상단 탭에서 "카카오 발송"을 누릅니다.
2
발송 문자 확인
노란색 박스에 오늘 날짜, 카테고리별 메뉴 목록, 주문 링크가 자동으로 완성되어 있습니다.
3
📋 텍스트 복사하기 버튼 클릭
버튼을 누르면 "복사 완료!"가 표시됩니다. 이제 카카오에 붙여넣기만 하면 됩니다.
💡 맨 아래 주문 링크가 자동으로 포함됩니다. 따로 입력하실 필요 없어요!
8

카카오 채널에 붙여넣기

복사한 문자를 카카오 비즈니스 채널에 붙여넣어 발송합니다.

1
카카오 비즈니스 채널 관리자 페이지 열기
카카오톡 채널 관리자 페이지에 접속합니다.
2
메시지 입력창 클릭 후 붙여넣기
메시지 입력창을 클릭한 후 Ctrl+V (PC) 또는 길게 눌러 붙여넣기 (스마트폰)
3
발송
내용을 확인하고 발송합니다.
9

카카오 텍스트 가져오기

이미 카카오에 보낸 메뉴 텍스트를 붙여넣으면 자동으로 메뉴 목록으로 변환됩니다.

1
"가져오기" 탭 클릭
상단 탭에서 "가져오기"를 누릅니다.
2
카카오 텍스트 붙여넣기
카카오톡에서 보낸 메뉴 메시지를 복사해서 입력창에 붙여넣기합니다.
형식: 한우육전 10 처럼 메뉴명 + 금액(천원)이면 됩니다.
3
🔍 메뉴 자동 인식하기 클릭
텍스트를 분석해서 메뉴 목록으로 변환합니다. 카테고리도 자동으로 추정됩니다.
4
카테고리 확인 후 가져오기
인식된 메뉴 목록을 확인하고, 카테고리가 잘못된 경우 수정합니다.
5
✅ 메뉴 관리로 가져오기 클릭
메뉴 목록에 추가됩니다. 같은 이름의 메뉴는 중복 없이 자동으로 처리됩니다.
💡 매일 새로 입력하기 귀찮으시면 이 기능을 쓰세요!
기존에 카카오로 보내시던 텍스트를 그대로 붙여넣기만 하면 됩니다.
10

메뉴 창고 저장하기

메뉴를 메뉴 창고에 저장하면 나중에 다시 불러올 수 있습니다.

1
"카카오 발송" 탭으로 이동
상단 탭에서 "카카오 발송"을 누릅니다.
2
메뉴 창고에 저장 버튼 클릭
오늘 날짜로 메뉴가 저장됩니다. "저장되었습니다!" 메시지가 뜨면 완료입니다.
💡 메뉴를 추가·수정할 때마다 자동으로 임시 저장됩니다.
메뉴 창고 저장은 나중에 불러오기 위한 별도 보관입니다.
11

메뉴 창고 보기

지난 날 메뉴를 불러와서 오늘 메뉴로 재활용할 수 있습니다.

1
"메뉴 창고" 탭 클릭
상단 탭에서 "메뉴 창고"를 누르면 저장된 날짜 목록이 표시됩니다.
2
불러올 날짜 클릭
원하는 날짜 카드를 클릭하면 그 날의 메뉴가 바로 표시됩니다.
지금 작업중 표시가 현재 선택된 날짜입니다.
3
메뉴 관리 탭으로 자동 이동
선택한 날짜의 메뉴가 메뉴 관리 탭에 표시됩니다. 수정 후 오늘 날짜로 다시 저장하세요.
💡 매주 비슷한 메뉴를 판매하신다면 지난주 메뉴를 불러온 뒤 일부만 수정하세요. 시간이 훨씬 절약됩니다!
